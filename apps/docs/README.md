# docs

Next.js application for project documentation/reference pages.

## Commands

```bash
npm -w apps/docs run dev
npm -w apps/docs run lint
npm -w apps/docs run check-types
npm -w apps/docs run build
```

## Notes

- Uses shared configs from `@repo/eslint-config` and `@repo/typescript-config`.
- Can consume shared UI from `@repo/ui`.
